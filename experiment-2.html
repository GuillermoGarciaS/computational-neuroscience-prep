Experiment #2 — Entropy, Content, and Representation

The first experiment measured how text size grows. This second experiment asks a harder question: if two texts have the same length, can one still contain more information than the other?

Introduction

Central question

Given two texts of equal length, which one contains more information—and how can we measure that without confusing information with meaning?

Why this matters

Length is not the same as information

A repetitive text, a meaningful sentence, and a random string can have the same number of characters. Their file size may be similar, but their predictability can be very different.

Open details ↓

This experiment studies information at the level of symbol distributions. It does not claim to measure semantic meaning, intelligence, or consciousness. That limitation is part of the point.

Theory

Shannon entropy treats a text as a source of symbols. First, we count how often each symbol appears. Then we convert those counts into probabilities. Finally, we calculate the average surprise of the source.

H(X) = - Σ p(x) log₂ p(x)

In this experiment, H(X) is the entropy of the selected text representation. The symbol x may be a character, a word, or a pair of characters depending on the mode. The probability p(x) is estimated from the text itself.

Method

Controls

Both texts are trimmed to the same character length before analysis. This prevents the longer text from winning simply because it contains more symbols.

Variables

Independent: type of content Controlled: length Dependent: entropy

Interpretation

If a text has high entropy, its symbols are harder to predict from frequency alone. If it has low entropy, its symbols are more repetitive or unevenly distributed.

The important twist is that a random text can score higher than a meaningful sentence. This shows why Shannon entropy is powerful but limited: it measures uncertainty, not semantic meaning.

Limits

Main limitation

Entropy is exact only after a representation has been chosen. Character entropy, word entropy, and bigram entropy are not interchangeable. The same document can look different depending on the lens used to measure it.

Core idea

Same length does not mean same information. Distribution matters.

Warning

High entropy can mean randomness, not depth.
