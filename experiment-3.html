Experiment #3 — Signals Over Time

The previous experiments treated information mostly as a static distribution. This experiment adds time. Instead of asking only how symbols are distributed, it asks how structure appears or disappears depending on the observation window.

Introduction

Central question

If a signal changes over time, does entropy measure the signal itself—or the signal as seen through a particular window?

Why this matters

Time changes what counts as structure

A signal can look random at a small scale and structured at a larger scale. The reverse can also happen: a signal can look simple locally, but complex when observed across a longer interval.

Theory

A signal can be described as a sequence of values across time. In this page, the signal is deliberately simple: a binary sequence made of 0s and 1s. This is not meant to simulate the brain directly. It is a controlled model for studying temporal structure.

S = [x₁, x₂, x₃, ..., xₜ]

Here, S is the signal and each xₜ is the value of the signal at time t. A binary signal has only two possible values, but the order of those values can still create patterns.

H(Wₜ) = - Σ p(x) log₂ p(x)

Instead of calculating entropy for the entire signal at once, this experiment calculates entropy inside a sliding window. The window moves across the signal, producing a local entropy value at each position.

Wₜ = [xₜ, xₜ₊₁, ..., xₜ₊ₖ₋₁]

The value k is the window size. A small k focuses on local structure. A large k captures broader structure, but may also smooth away short irregularities.

Examples

Random signal

High uncertainty

A random signal tends to produce high entropy because each window contains a relatively unpredictable mix of 0s and 1s.

Periodic signal

Scale-dependent structure

A periodic signal may look simple in one window and balanced in another. Its entropy depends strongly on how much of the cycle the window contains.

Noisy pattern

Structure plus disturbance

Noise pushes entropy upward, but it does not always erase the pattern immediately. The original structure can still shape the resulting entropy curve.

Method

Variables

Independent: signal type Independent: window size Independent: noise level Dependent: local entropy

Procedure

Step Action Reason
1 Generate a binary signal. Keep the system simple enough to inspect.
2 Move a window across the signal. Measure local structure instead of only global frequency.
3 Calculate entropy inside each window. Track how uncertainty changes over time.

Interpretation

The important result is that entropy is not only a property of the signal. It is also a property of the way the signal is observed. Changing the window size changes the scale at which structure becomes visible.

This matters for the larger project because perception also depends on scale, resolution, and transformation. A system does not encounter raw reality all at once. It samples, filters, compresses, and organizes signals through constraints.

Limits

Main limitation

This experiment only measures the balance of 0s and 1s inside each window. It does not yet detect higher-order patterns, long-range repetition, causality, or meaning. A sequence can have the same local entropy as another sequence while having a very different temporal organization.

Core idea

Entropy depends on the signal and the observation window.

Warning

A window can reveal structure, but it can also hide it.
