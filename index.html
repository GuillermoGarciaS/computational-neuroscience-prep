Framework

The project moves in three layers: measuring uncertainty, observing how structure emerges, and testing whether structured output is enough to count as experience.

Layer 1

Entropy

Measure uncertainty, repetition, randomness, and how information can be quantified without assuming meaning.

Layer 2

Representation

Track how systems transform input into structure through regression, comparison, and constrained modeling.

Layer 3

Limits

Ask what remains absent once uncertainty has been reduced: interpretation, subjectivity, and lived experience.
