From Entropy to Experience

A long-term portfolio of computational experiments on information, uncertainty, pattern detection, representation, and the limits of explaining subjective experience.

Start with Experiment 1 → View learning path
Research compass

A guided path for studying how information becomes structure, how structure becomes detectable, and why detection is still not the same as experience.

Starting point

Entropy and predictability

Long-term aim

Representation without overclaiming

Core Question

To what extent can reducing uncertainty in computational systems explain structured representation, and where does that explanation fail?

Approach

Each page isolates a measurable idea: entropy, distribution, time windows, detection, and then synthesis.

Scope

This project does not claim to explain consciousness. It maps what computational explanation can and cannot capture.
